# Prompt Templates

Gemini CLI に渡すプロンプトのテンプレート集。

## コードレビュー

```
以下のコードをレビューしてください：
- バグや潜在的な問題
- パフォーマンスの改善点
- セキュリティの懸念
- コードの可読性・保守性
```

**使用例:**
```bash
cat {ファイルパス} | gemini -p "以下のコードをレビューしてください：バグや潜在的な問題、パフォーマンスの改善点、セキュリティの懸念、コードの可読性・保守性" -m gemini-3-pro-preview
```

---

## 論文・学術文章チェック

```
以下の文章を学術的な観点からレビューしてください：
- ロジックの一貫性
- 文法・表現の改善点
- 引用や根拠の妥当性
```

**使用例:**
```bash
cat {論文ファイルパス} | gemini -p "以下の文章を学術的な観点からレビューしてください：ロジックの一貫性、文法・表現の改善点、引用や根拠の妥当性" -m gemini-3-pro-preview
```

---

## セキュリティ監査

```
以下のコードをセキュリティの観点から監査してください：
- 認証・認可の問題
- インジェクション攻撃（SQLi, XSS等）への脆弱性
- 機密情報の取り扱い
- 依存ライブラリのリスク
```

---

## アーキテクチャレビュー

```
以下のコードのアーキテクチャをレビューしてください：
- 設計パターンの適切さ
- モジュール間の依存関係
- スケーラビリティの懸念
- テスタビリティ
```

---

## JSON 出力が必要な場合

`--output-format json` を追加する：

```bash
cat {ファイルパス} | gemini -p "{プロンプト}" -m gemini-3-pro-preview --output-format json
```
