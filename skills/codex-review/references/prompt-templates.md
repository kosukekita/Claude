# Prompt Templates

Codex CLI に渡すプロンプトのテンプレート集。
`codex review` の場合は引数として、`codex exec` の場合はパイプ後の引数として使用する。

## コードレビュー

```
以下のコードをレビューしてください：
- バグや潜在的な問題
- パフォーマンスの改善点
- セキュリティの懸念
- コードの可読性・保守性
日本語で回答してください。
```

**git diff 使用例:**
```bash
codex review --uncommitted "以下のコードをレビューしてください：バグや潜在的な問題、パフォーマンスの改善点、セキュリティの懸念、コードの可読性・保守性。日本語で回答してください。"
```

**ファイル使用例:**
```bash
cat {ファイルパス} | codex exec "以下のコードをレビューしてください：バグや潜在的な問題、パフォーマンスの改善点、セキュリティの懸念、コードの可読性・保守性。日本語で回答してください。"
```

## 論文・学術文章チェック

```
以下の文章を学術的な観点からレビューしてください：
- ロジックの一貫性
- 文法・表現の改善点
- 引用や根拠の妥当性
日本語で回答してください。
```

**使用例:**
```bash
cat {論文ファイルパス} | codex exec "以下の文章を学術的な観点からレビューしてください：ロジックの一貫性、文法・表現の改善点、引用や根拠の妥当性。日本語で回答してください。"
```

---

## セキュリティ監査

```
以下のコードをセキュリティの観点から監査してください：
- 認証・認可の問題
- インジェクション攻撃（SQLi, XSS等）への脆弱性
- 機密情報の取り扱い
- 依存ライブラリのリスク
日本語で回答してください。
```

---

## アーキテクチャレビュー

```
以下のコードのアーキテクチャをレビューしてください：
- 設計パターンの適切さ
- モジュール間の依存関係
- スケーラビリティの懸念
- テスタビリティ
日本語で回答してください。
```

---

## PR前の最終確認（git diff 向け）

```bash
# 未コミット変更の総合レビュー
codex review --uncommitted "このPR前の変更について総合的にレビューしてください。マージしても問題ないか、懸念点があれば日本語で教えてください。"

# ブランチ差分のレビュー
codex review --base main "main ブランチとの差分をレビューしてください。日本語で回答してください。"
```
