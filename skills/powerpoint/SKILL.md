# PowerPoint スキル

## 重要ルール

- **Marp 使用禁止**: スライド作成に Marp は使用しない。python-pptx で直接 .pptx を生成すること。
- **python-rules スキルを参照**: Python 実行環境は uv + PEP 723 インラインスクリプトで管理。
- **1スライド1メッセージ**: 各スライドに伝えたいメッセージは1つに絞る。

## デザイン設定

### カラーパレット

| 役割 | 色 | 用途 | 配分 |
|------|-----|------|------|
| 背景 | `#F9F9F9` | スライド背景（無地白） | — |
| ベースカラー | `#1A1A1A` | 本文テキスト | 70% |
| メインダーク | `#00337D` | タイトル・重要見出し | メイン計25% |
| メイン | `#004BB1` | スライド見出し・装飾 | |
| メインライト | `#E3F2FD` | 背景ハイライト・淡い塗り | |
| アクセント | `#FEDA16` | 重要ポイント強調（5%以内） | 5% |
| 注釈 | `#666666` | 出典・補足テキスト | — |

**色の原則**: メインカラー3色（`#00337D` / `#004BB1` / `#E3F2FD`）+ アクセント1色（`#FEDA16`）で統制。アクセントは本当に強調したい箇所のみに限定使用。

### タイポグラフィ

| 要素 | フォント | サイズ | 色 |
|------|---------|--------|-----|
| タイトル | Segoe UI / Meiryo | 50pt | `#00337D` |
| スライド見出し | Segoe UI / Meiryo | 35pt | `#004BB1` |
| 本文 | Segoe UI / Meiryo | 25pt | `#1A1A1A` |
| 注釈・出典 | Segoe UI / Meiryo | 18pt | `#666666` |

**フォントの原則**: 全スライドで同じフォントファミリーを統一。英数字は Segoe UI 優先、日本語は Meiryo。

### レイアウト

- **スライドサイズ**: 16:9（ワイド）
- **余白**: 上下左右 0.5インチ以上
- **オブジェクト配置**: テキスト・図表の位置をスライド間で揃える（グリッド・整列機能を活用）
- **情報階層**: フォントサイズの大小で階層を明示（タイトル > 見出し > 本文 > 注釈）

## 学会発表スライドの構成

### 標準構成（口頭発表）

| # | スライド種別 | 内容 | 枚数目安 |
|---|-------------|------|----------|
| 1 | タイトル | 演題・著者・所属・学会名 | 1 |
| 2 | 利益相反 | COI 開示（学会要件に応じて） | 1 |
| 3 | 背景 | 一般的な話題 → 専門的内容へ段階的に | 2-3 |
| 4 | 目的 | 本研究の目的・意義を明確に | 1 |
| 5 | 方法 | 実験手法・調査方法・対象 | 2-4 |
| 6 | 結果 | データ・グラフ・図表で視覚的に提示 | 3-6 |
| 7 | 考察 | 結果の解釈・先行研究との比較 | 2-3 |
| 8 | 結論 | まとめ・今後の課題 | 1 |

**枚数目安**: 発表時間(分) x 1.5〜2枚（10分発表 → 15〜20枚）

### 聞き手への配慮

- 専門外の参加者を想定し、専門用語は初出時に簡潔に説明
- 背景スライドは一般的な話題から専門的内容へ段階的に展開
- 補足情報はスライドに詰め込まず、発表原稿に含める

## スライド種別テンプレート

### タイトルスライド
- 演題: **50pt**、メインダーク（`#00337D`）、中央揃え
- 著者・所属: **25pt**、TEXT色
- 学会名・日付: **18pt**、`#666666`
- 背景: BG色無地

### コンテンツスライド
- スライドタイトル: **35pt**、メイン（`#004BB1`）、左揃え
- 本文箇条書き: **25pt**、TEXT色
- 図表がある場合: テキストと図表の領域を明確に分離

### 図表スライド
- スライドタイトル: **35pt**、メイン（`#004BB1`）
- 図表: スライド中央〜右寄り、十分なサイズ
- 図表タイトル・軸ラベル: **18pt** 以上（会場後方からの視認性確保）
- 出典: **18pt**、`#666666`、スライド下部

### セクション区切りスライド
- 見出し: **35pt**、白文字
- 背景: メインダーク（`#00337D`）

## 図表・グラフのルール

- 文字だけの説明は避け、データは必ず視覚的に表現
- グラフの軸ラベル・凡例は会場後方から読めるサイズ（18pt以上）
- 表は罫線を最小限にし、行間を広くとって読みやすく
- 1スライドに図表は原則1つ（比較時は最大2つ）
- 強調したいデータポイントにはアクセント色（`#FEDA16`）を使用

## python-pptx 実装ガイド

### 基本セットアップ（PEP 723 インラインスクリプト）

```python
# /// script
# requires-python = ">=3.11"
# dependencies = ["python-pptx"]
# ///
from pptx import Presentation
from pptx.util import Inches, Pt
from pptx.dml.color import RGBColor

# カラー定数
COLOR_BG         = RGBColor(0xF9, 0xF9, 0xF9)
COLOR_TEXT       = RGBColor(0x1A, 0x1A, 0x1A)
COLOR_MAIN_DARK  = RGBColor(0x00, 0x33, 0x7D)
COLOR_MAIN       = RGBColor(0x00, 0x4B, 0xB1)
COLOR_MAIN_LIGHT = RGBColor(0xE3, 0xF2, 0xFD)
COLOR_ACCENT     = RGBColor(0xFE, 0xDA, 0x16)
COLOR_NOTE       = RGBColor(0x66, 0x66, 0x66)
```

### スライドサイズ（16:9）

```python
prs = Presentation()
prs.slide_width  = Inches(13.33)
prs.slide_height = Inches(7.5)
```

### 背景色設定

```python
def set_slide_background(slide, color: RGBColor):
    fill = slide.background.fill
    fill.solid()
    fill.fore_color.rgb = color
```

### フォント設定

```python
def apply_font(run, size_pt: int, bold=False, color=COLOR_TEXT):
    run.font.name  = "Segoe UI"   # 英数字
    run.font.size  = Pt(size_pt)
    run.font.bold  = bold
    run.font.color.rgb = color
    # 日本語: PowerPoint が Meiryo にフォールバック
```
